# IPIPO 黑客松开发计划 (6小时MVP)

## 🎯 黑客松MVP目标
基于README项目说明，专注实现影响力预售平台的**核心交互流程**：
1. 创作者创建活动 → 2. 买家浏览购买 → 3. 持券人核销 → 4. 创作者提现
